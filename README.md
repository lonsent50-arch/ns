# Novel Studio - 专业小说写作工具

一款面向商业级使用的本地小说写作工具，支持 AI 辅助创作。

## 功能特性

### 核心写作
- 📄 **章节管理**：创建/删除/排序章节，自动保存
- ✍️ **专业编辑器**：深色护眼主题，支持大数据量文本
- 💾 **自动保存**：3秒防抖自动保存，Ctrl+S 手动保存
- 📊 **写作统计**：实时字数统计、今日写作量、章节数

### 角色系统
- 👥 **角色卡片**：姓名、性别、年龄、性格、背景、目标、外貌
- 📋 **角色列表**：可视化卡片展示，一键编辑/删除

### 情节大纲
- 🗺️ **树形大纲**：支持多层级大纲节点（拖拽排序）
- 📝 **节点编辑**：每个大纲节点独立编辑内容

### AI 辅助创作（DeepSeek API）
- 💬 **AI 对话**：通用 AI 助手，讨论剧情/角色/构思
- ✍️ **AI 续写**：基于当前内容智能续写，自动插入编辑器
- ✨ **AI 润色**：提升文笔，支持指定润色方向
- 🎭 **去 AI 味**：将 AI 生成文字改写为更自然的"人味"文字
- 💡 **头脑风暴**：创作瓶颈时获取灵感建议

### 导出与发布
- 📘 **导出 Word (.docx)**：保留章节结构，适合打印/分享
- 📄 **导出纯文本 (.txt)**：通用格式，兼容所有设备

### 写作目标
- 🎯 **每日目标**：设定每日字数目标，追踪完成率
- 📈 **长期目标**：设定总字数目标

## 快速启动

```bash
cd novel-studio
chmod +x start.sh
./start.sh
```

或者手动启动：

```bash
pip3 install flask flask-cORS python-docx reportlab markdown
python3 app.py
```

然后浏览器打开：`http://127.0.0.1:5050`

## 使用说明

1. **创建项目**：点击左下角「＋ 新建项目」，输入项目名称
2. **创建章节**：在写作区点击「＋ 章节」，开始写作
3. **AI 功能**：左侧导航栏切换到对应 AI 功能 Tab
4. **角色/大纲**：左侧导航进入对应管理页面
5. **导出**：顶部工具栏或「导出作品」Tab

## 技术架构

- **后端**：Flask (Python 3.12) + SQLite
- **前端**：原生 HTML/CSS/JS，无框架依赖
- **AI 引擎**：DeepSeek API（内置 Key，可替换）
- **数据存储**：项目数据存储在 `projects/` 目录，每项目独立 SQLite 数据库

## 项目结构

```
novel-studio/
├── app.py              # Flask 后端主程序
├── start.sh           # 一键启动脚本
├── projects/          # 项目数据目录（自动创建）
├── static/
│   ├── index.html    # 主页面（Scrivener 风格三栏布局）
│   ├── css/style.css # 深色主题样式
│   └── js/app.js    # 前端核心逻辑
└── README.md
```

## 注意事项

- 所有数据**本地存储**，不会上传云端
- DeepSeek API Key 内置在 `app.py` 中，如需替换请编辑该文件
- 建议用 Chrome/Safari 最新版浏览器
- 支持百万字小说（已做上下文截断处理，避免 API 400 错误）

## 商业使用

本工具完全开源，可自由修改、分发、商用。
